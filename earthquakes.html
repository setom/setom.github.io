USGS Live GeoJSON: 2.5m Earthquakes over the last 7 days

Hover over an Earthquake for details

Earthquake
size indicates magnitude

Data from USGS. Map updates every 5 minutes

This map is pulling live data from the USGS earthquake GeoJSON server. The map uses the Google Maps API to deliver a stylized, simply to read map that displays earthquakes around the world from the last 7 days. Click earthquake points to view additional information, or to follow USGS links to view further details about the event.


Global Cruise Ships Live KMZ: Continual Cruise Ship Locations & Vectors

Ships are identified by the logo of their carrier
Click a ship for further details

Data from Sea Scanner. KML Data Updated by Seascanner

This map is pulling live data about the locations, trajectories and other details of passenger cruise ships. The data is from a KMZ source that is unpacked and displayed using the Google Maps API. The basemap has been stylized by me, however the point data inherits the stylization from the source.


Snowfall Probability Live KML: Probability of >4 Inches of Snow in the Next 24 Hours

Data from NCEP-NOAA. Data Updated by NCEP-NOAA

This map is displaying live data from NOAA-NCEP's forecasting tools. The polygons on the map depict areas where more than 4 inches of snow is likely. The data is a KML source displayed using the ESRI ArcGis JS3.22 library. The basemap has been stylized, however KML files inherit their stylization from the source.
