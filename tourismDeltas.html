Changes in Tourist Visits in 2004 - 2014

This interactive map displays the percent change of incoming tourists to individual countries. Incoming tourist visa counts were collected for 2004 and 2014. Countries with insufficient data are excluded from this analysis.

This map uses Leaflet, and incorporates data from the UN Data portal. Data was cleaned by me, joined to global polygons and prototyped in QGis. The data was converted to GeoJSON for use in conjunction with the Leaflet API
